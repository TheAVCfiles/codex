# Frequently Asked Questions

## How does Sentient Cents work?
Each verified event is scored and minted into cents that appear on the ledger demo dashboard.

## Where is my data stored?
Engagement metadata is stored in Cloudflare KV/D1, and normalized via the CSV tools before distribution.

## How do I request deletion?
Use the contact form embedded in the microsite or email privacy@yourorg.com for immediate action.
