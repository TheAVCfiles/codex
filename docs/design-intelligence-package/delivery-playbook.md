# Delivery Playbook (White-Glove, Low-Friction)

## Phase 0: Intake (24–48 hours)

- Confirm scope, timeline, and sponsor.
- Collect required inputs using the kickoff brief.
- Establish a shared workspace and file taxonomy.

**Exit criteria:** scope signed, inputs received, owners identified.

## Phase 1: Signal Capture (Week 1)

- Review existing artifacts.
- Map stakeholders and decision pathways.
- Identify missing inputs and request only what is essential.

**Exit criteria:** signal inventory logged, gaps documented, risk register created.

## Phase 2: Analysis & Synthesis (Week 2–3)

- Build current-state experience map.
- Identify opportunity clusters.
- Draft decision map and tradeoffs.

**Exit criteria:** internal review completed, draft summary ready.

## Phase 3: Alignment & Validation (Week 3–4)

- Facilitate a single alignment session.
- Present interim findings and confirm decision owners.
- Adjust priorities based on governance constraints.

**Exit criteria:** alignment achieved, decisions logged.

## Phase 4: Delivery (Week 4+)

- Deliver the executive summary and implementation brief.
- Provide a handoff walkthrough.
- Archive artifacts in the designated repository.

**Exit criteria:** sign-off from sponsor, handoff acknowledged.

## SLAs and cadence

- Weekly status update (asynchronous)
- One live session per phase (max)
- 48-hour turnaround for critical questions

## Escalation protocol

- Any blocker over 72 hours triggers sponsor escalation.
- Scope changes are logged and re-approved.
