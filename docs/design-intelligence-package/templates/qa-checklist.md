# QA Checklist (Delivery Readiness)

## Scope & alignment

- [ ] Scope confirmed with sponsor
- [ ] Decision owners identified
- [ ] Constraints documented

## Evidence & traceability

- [ ] Recommendations link to evidence
- [ ] Assumptions documented
- [ ] Risk register reviewed

## Deliverable quality

- [ ] Executive summary approved
- [ ] Decision map validated
- [ ] Implementation brief complete

## Delivery hygiene

- [ ] Final artifacts archived
- [ ] Handoff completed
- [ ] Next steps documented
