## Proposed Archival Folder Structure

**Goal:** Preserve provenance with version control and consistent naming.

1. **1_Performance-Choreography/**

   - Subfolders by year/production (e.g., `2008_HS_Midsummer_Nights_Dream/`).
   - Example file: `2008_Titania_Midsummer_playbill.pdf`

2. **2_Educational-Roles-Curricula/**

   - Subfolders by institution/program (e.g., `2019-2022_YourAfterSchool/`).
   - Include syllabi, lesson plans, testimonials.

3. **3_Tech-AI-Consulting/**

   - Subfolders like `2024_RWS-Consulting/`.
   - Include design docs, code snippets, project charters.

4. **4_Public-Authorship-Media/**

   - Blog posts, poetry publications, media features.
   - Organized by format/date.

5. **5_Legal-Financial/**

   - Invoices, timesheets, contracts, statements.
   - Example: `2025-11-13_CPS_interference_invoice.pdf`

6. **6_Professional-Profiles_Resumes/**

   - Resume/CV versions, bios, LinkedIn screenshots, interactive resume exports.

7. **7_Institutional-Lineage-Research/**
   - Lineage dossiers, institutional histories, archival research.

**Naming convention:** `{Year}_{InstitutionOrProject}_{Description}_{Role}.{ext}`  
**Master index:** Spreadsheet or ledger listing date, role, institution, artifact type, verification status.
