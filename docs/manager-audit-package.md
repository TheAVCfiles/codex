## III. Forensic Audit Checklist (“Audit the Loop”)

This sequence defines required tasks and expected outputs.

### 1) Revenue & Tax Reconciliation (Ghost 1099 Neutralization)

- Pull IRS Wage & Income Transcripts (2019–2025) via IRS.gov “Get Transcript Online” or Form 4506-T.
- Match the $17,841 1099-NEC (2021) to actual bank deposits for the same period.
- **Output:** Table showing claimed income vs. net deposits retained.
- **Expected discrepancy flag:** If net deposits ≈ $0, potential tax reporting irregularity.

### 2) Asset Stripping Audit ($55,000+ Floor)

- Export CSV/PDF from brokerage account(s) showing all liquidations (2020–2022).
- Export bank statements for the same period.
- Match each liquidation date + amount to outflows within 72 hours.
- **Output table columns:**
  - Sale Date
  - Amount
  - Deposit Account
  - Outflow Date
  - Amount
  - Merchant
  - Beneficiary (household/business)
- **Expected result:** Pattern of proceeds flowing into subject-controlled expenses/property improvements.

### 3) Professional Inventory & Conversion Audit

- List every retained/distributed item (gowns, costumes, choreography notebooks, production tools).
- Assign replacement value (current market replacement, not original cost).
- Document last known location/status.
- **Output:** Inventory Ledger.
- **Expected result:** Quantifiable conversion/tortious interference damages.
