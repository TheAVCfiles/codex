# Export Checklist (Next Pull)

## Banking / Cashflow
- [ ] Bank statements (monthly PDFs) covering full target period.
- [ ] Transaction CSV exports (debit/credit detail).
- [ ] Zelle transaction history export (if available).

## Brokerage / Securities
- [ ] Brokerage monthly statements (PDF).
- [ ] Realized gain/loss export CSV.
- [ ] Transfer and withdrawal logs.

## Tax / Wage
- [ ] 1099 forms by year (PDF scans + typed summary).
- [ ] Any W-2 or payroll statements (if present).
- [ ] Venmo/PayPal annual summaries.

## Communications
- [ ] iMessage full thread exports (not screenshots only).
- [ ] Email thread exports (EML/PDF) with full headers for key events.
- [ ] Calendar exports showing work schedule and control indicators.

## Receipts / Supporting
- [ ] Receipts tied to alleged extraction events.
- [ ] Contracts, offer letters, or role announcements.
- [ ] Storage / travel / production expense receipts.

## Intake Rules
- Preserve originals in `CASE_CAPSULE/evidence_raw/` untouched.
- Add one ledger entry per artifact in `case_capsule_master.yaml`.
- Use evidence ID format `EVT-YYYYMMDD-###` or `EVT-UNKNOWN-###`.
- Keep descriptions neutral and factual.
