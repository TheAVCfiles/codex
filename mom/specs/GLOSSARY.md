# Glossary (M.O.M.)

- **MemNode™**: governed memory object (encrypted payload + consent + salience).
- **Corridors™**: typed relationships with traversal constraints.
- **Coda™**: append-only continuity log; no overwrite guarantees.
- **StagePort™**: policy enforcement boundary; fail-closed.
- **Salience**: priority weight (0–1) based on user signals + sentiment.
- **Masked**: memory available for internal recall but redacted in exposed contexts.
- **Nudge Constitution**: strict rules defining when proactive surfacing is allowed.
