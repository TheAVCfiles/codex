# Code of Conduct

M.O.M. is built for systems that touch identity, memory, and safety. This is not a playground for cruelty.

## We expect

- Respectful collaboration
- Clear technical critique without personal attacks
- Good-faith handling of sensitive topics (privacy, abuse, consent, safety)

## We will not tolerate

- Harassment, threats, or intimidation
- Doxxing or attempts to reveal personal data
- Sexualized or violent harassment
- “Just trolling” as a personality

## Reporting

Report incidents to: [YOUR-CONDUCT@DOMAIN.COM]

## Enforcement

Violations may result in warnings, content removal, or bans from the project spaces.
